# Cortex Charter

Remram Cortex exists to turn runtime evidence and source material into durable, inspectable, layered knowledge without collapsing every job into one memory substrate.

## Core Commitments

Cortex is committed to:

- explicit policy composition
- hot working continuity that stays fast and pruneable
- durable semantic memory with lineage and reconciliation
- operational knowledge that can evolve before publication
- canonical publication only when canon is actually warranted
- fewer standing services and clearer authority boundaries

## System Boundary

OpenClaw executes.

Cortex:

- governs policy composition
- governs Layer 2 notions and high-signal continuity
- governs durable-memory promotion into Layer 3
- governs how Layer 3 helps organize Layer 4
- governs how Layer 4 and Layer 5 stay aligned when canon exists

## Layer Commitments

### Policy

Layer 1 is small and custom.

It owns:

- role and mode composition
- tool-use rules
- approval posture
- escalation posture
- prompt-budget discipline
- mutable preference-policy

### Working Memory

Layer 2 is hot working continuity.

It uses:

- OpenClaw sessions and runtime continuity
- `QMD` for hot retrieval and notion storage
- boundary-triggered semantic processing in early phases
- a narrow Mamba stream later for high-signal listening

### Durable Memory

Layer 3 is durable semantic truth.

It owns:

- concepts
- identities
- relationships
- support
- supersession
- invalidation

### Operational Knowledge

Layer 4 is the operational knowledge authority.

It owns:

- incubation workspaces
- working synthesis bodies
- reference-derived operational knowledge
- decomposed artifact knowledge

### Canonical Artifacts

Layer 5 is canonical publication truth when publication-grade authorship and revision are warranted.

It is not the default destination for every useful idea or external reference.

## Why This Exists

Without Cortex, systems drift toward:

- transcript-as-memory
- prompt-as-policy
- hot continuity pretending to be durable truth
- operational workspaces jumping to canon too early
- external references being treated like authored artifacts

Cortex exists to stop that drift.

## Delivery Posture

Phase 1 proves the spine without live `Mamba`.

After Phase 1, a simple decision gate determines whether `Mamba` should be pulled forward as a Phase `1.5` spike or remain deferred until Phase 3.
