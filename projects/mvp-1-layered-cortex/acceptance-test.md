# Acceptance Test

The MVP should prove the following end-to-end behaviors:

1. policy overlays affect runtime behavior without replacing OpenClaw session mechanics
2. the semantic checkpoint stream reduces prompt pressure on smaller-window models
3. high-signal notions can appear cross-thread as tentative memory
4. reconciliation promotes or rejects tentative memory correctly
5. artifact intake produces Layer 4 retrieval surfaces and preserves Layer 5 truth
6. oversight can inspect semantic checkpoints and intervene where required
