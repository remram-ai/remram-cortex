# Cortex v0.1 MVP Project

This folder holds the delivery artifacts for the first Cortex MVP.

The v0.1 slice is intentionally narrow:

- seed a baseline knowledge base through artifact intake
- persist Cortex-managed source evidence through artifact storage
- prove the live write path through Quick Capture
- prove the live read path through ordinary chat with knowledge injection
- retrieve imported and captured knowledge through one Cortex model

The storage contract is provider-based even if the first implementation uses only one backend.
