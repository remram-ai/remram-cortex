# Layered Cortex Stack Context Pack

This pack is the AI-oriented briefing set for the current Cortex stack.

It is optimized for:

- NotebookLM
- audio briefings
- agent onboarding
- fast architectural synthesis

It deliberately restates the active design in stable repeated language so an AI system can build a coherent mental model without also ingesting the archived decision churn.

## Pack Goal

Explain the chosen Cortex stack topic by topic:

1. what Cortex is
2. how the layers work
3. how OpenClaw fits
4. what the High-Signal Mamba Stream is
5. how durable memory works
6. how knowledge and artifacts work
7. how reflection, oversight, and reconciliation work
8. how the runtime and build sequence fit together
