# Project Plan

## Epic Order

1. Policy and OpenClaw integration
2. QMD Layer 2 and High-Signal Mamba stream
3. Graphiti + Neo4j durable memory
4. Operational knowledge and canonical artifact boundaries
5. Reflection, Dream, oversight, and promotion readiness

## Why This Order

- Layer 1 and Layer 2 must be stable before the system can feed good durable memory.
- QMD and the high-signal stream must exist before notion handling is real.
- Layer 3 must be stable before it can help organize Layer 4.
- Layer 4 must be clear before promotion to Layer 5 makes sense.
- reflection, Dream, and oversight should harden the system after the main flows exist.
