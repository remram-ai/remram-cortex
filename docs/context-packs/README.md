# Context Packs

`docs/context-packs/` holds AI-oriented briefing packages for the active Cortex architecture.

These packs are intentionally more redundant and more topic-organized than the main design docs.

Use them when you want:

- a NotebookLM source set
- an audio debrief
- a model-onboarding package
- a smaller self-contained reading set than the whole repository

## Active Packs

- [Layered Cortex Stack](layered-cortex-stack/README.md)

## Rule

These packs should reflect the active chosen architecture only.

Archived contender analysis, rejected stack options, and older repository postures belong in the top-level [archive/](../../archive/README.md), not here.
